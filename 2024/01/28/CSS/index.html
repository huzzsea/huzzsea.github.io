第一节 CSS简介

  • CSS：全称 Cascading Style Sheets，即层叠样式表，样式定义如何显示 HTML 元素，解决了内容与表现分离的问题

  • CSS 规则：由两个主要的部分构成：选择器，以及一条或多条声明

    • 选择器通常是需要改变样式的 HTML 元素
    • 每条声明由一个属性和一个值组成，属性是需要设置的样式属性，每个属性有一个值，属性和值被冒号分开

  • CSS 注释：以 /* 开始, 以 ***/** 结束

第二节 CSS引用方式

  • 内联定义方式：指定各标签 style 属性

  • 样式块：使用 style 标签

  • 外部引用：引入外部独立的 CSS 样式文件

  • 优先级规则：行内样式 > 内部样式 = 外部样式

第三节 选择器

3.1 基本选择器

  • 通配选择器：选中所有的 html 元素

  • 元素选择器：为页面中某类元素统一设置样式

  • 类选择器：根据元素的 class 值选择一类元素

  • ID选择器：根据元素的 id 值选择某一特定元素

3.2 复合选择器

  • 交集选择器：选中同时符合多个条件的元素

  • 并集选择器：选中多个选择器对应的元素

  • 后代选择器：选中符合条件的后代元素

  • 子代选择器：选中符合条件的子代元素

  • 兄弟选择器：选中符合条件的兄弟元素

    • 相邻兄弟选择器

    • 通用兄弟选择器

  • 属性选择器：选中属性值符合一定条件的元素

    • 选中具有某个属性的元素

    • 选中包含某个属性，且属性值等于指定值的元素

    • 选中包含某个属性，且属性值以指定的值开头的元素

    • 选中包含某个属性，且属性值以指定的值结尾的元素

    • 选择包含某个属性，属性值包含指定值的元素

  • 伪类选择器：选中特殊状态的元素

    • 动态伪类：遵循 link、visited、hover、 active 的顺序

      动态伪类 描述
      :link 超链接未被访问的状态
      :visited 超链接访问过的状态
      :hover 鼠标悬停在元素上的状态
      :active 元素激活的状态
      :focus 获取焦点的元素

    • 结构伪类

      结构伪类 描述
      :first-child 所有兄弟元素中的第一个
      :last-child 所有兄弟元素中的最后一个
      :nth-child(n) 所有兄弟元素中的第 n 个
      :first-of-type 所有同类型兄弟元素中的第一个
      :last-of-type 所有同类型兄弟元素中的最后一个
      :nth-of-type(n) 所有同类型兄弟元素中的 第n个

    • 否定伪类：排除满足括号中条件的元素

    • UI伪类

      UI伪类 描述
      :checked 被选中的复选框或单选按钮
      :enable 可用的表单元素（没有 disabled 属性）
      :disabled 不可用的表单元素（有 disabled 属性）

  • 伪元素选择器：选中元素中的一些特殊位置

    伪元素 描述
    ::first-letter 选中元素中的第一个文字
    ::first-line 选中元素中的第一行文字
    ::selection 选中被鼠标选中的内容
    ::placeholder 选中输入框的提示文字
    ::before 在元素最开始的位置，创建一个子元素（必须用 content 属性指定内容）
    ::after 在元素最后的位置，创建一个子元素（必须用 content 属性指定内容）

3.3 选择器的优先级

  • 选择器的优先级：行内选择器 > ID 选择器 > 类选择器 > 元素选择器 > 通配选择器

第四节 CSS三大特性

  • 层叠性：如果发生了样式冲突，那就会根据一定的规则（选择器优先级），进行样式的层叠覆盖
  • 继承性：元素会自动拥有其父元素、或其祖先元素上所设置的某些样式
  • 优先级：样式按照一定规则为元素进行修饰

第五节 CSS常用属性

5.1 字体属性

属性 描述 属性值
font-size 字体大小 20px，30%
font-weight 字体粗细 normal，bold，400~700
font-family 字体 Microsoft YaHei，Hiragino Sans GB
font-style 字体样式 normal，italic(斜体)
font 简写属性 font:font-style font-weight font-size font-family

5.2 文本属性

属性 描述 属性值
color 颜色 #00FF00，rgb(red,green,blue)，rbga(red,green,blue,alpha)
text-align 文本对齐 left，center，right
text-decoration 文本修饰 none，underline(下划线)，overline(上划线)，line-through(删除线)
text-indent 首行缩进 2em，6px
line-height 行间距 20px

5.3 列表属性

属性 描述 属性值
list-style-type 列表符号 none，square(实心方块)，disc(圆形)，decimal(数字)，lower-roman(小写罗马字母)，upper-roman(大写罗马字母)，lower-alpha(小写字母)，upper-alpha(大写字母)
list-style-position 列表符号位置 inside，outside
list-style-image 自定义列表符号 url(图片位置)
list-style 简写属性 没有数量顺序要求

5.4 表格属性

属性 描述 属性值
table-layout 列宽度 auto，fixed(固定平均列宽)
border-spacing 单元格间距 2px
border-collapse 合并单元格边框 collapse(合并)，separate(不合并)
empty-cells 隐藏没有内容的单元格 show，hide
caption-side 表格标题位置 top，bottom

5.5 背景属性

属性 描述 属性值
background-color 背景颜色 transparent(默认)
background-image 背景图片 url(图片位置)
background-repeat 背景重复方式 repeat(重复)，repeat-x(水平重复)，repeat-y(垂直重复)，no-repeat(不重复)
background-position 背景图片位置 left，right，center
background 简写属性 没有数量顺序要求

5.6 鼠标属性

属性 描述 属性值
cuisor 光标样式 pointer(小手)，move(移动图标)，text(文字选择器)，crosshair(十字架)，wait(等待)，help(帮助)

第六节 盒子模型

6.1 CSS长度单位

  • 长度单位
    • px：像素
    • em：相对于字体大小的倍数
    • rem：相对于根字体的大小
    • **%**：相对父元素的比例

6.2 元素的显示模式

  • 元素的显示模式

    • 块元素：在页面中独占一行，从上到下排列，可以设置宽高
    • 行内元素：在页面中不独占一行，从左到右排列，无法设置宽高
    • 行内块元素：在页面中不独占一行，从左到右排列，可以设置宽高

  • 修改元素的显示模式：修改元素的 display 属性

    属性值 描述
    none 隐藏
    block 块元素
    inline 内联元素
    inline-block 行内块元素

6.3 盒子模型的组成

  • 盒子模型：将所有的 html 元素都看成一个盒子，所有的样式都是基于这个盒子

  • 盒子模型的组成

    • margin：外边距，盒子与外界的距离
    • border：边框，盒子的边框
    • padding：内边距，紧贴内容的补白区域
    • content：内容，元素中的文本或后代元素都是它的内容

  • 盒子模型的内容区

    属性 描述 属性值
    width 宽度 20px
    max-width 最大宽度 20px
    min-width 最小宽度 20px
    height 高度 20px
    max-height 最大高度 20px
    min-height 最小高度 20px

  • 盒子模型的内边距

    属性 描述 属性值
    padding-top 上内边距 10px
    padding-right 右内边距 10px
    padding-bottom 下内边距 10px
    padding-left 左内边距 10px
    padding 简写属性 10px 10px 10px 10px，10px

  • 盒子模型的边框

    属性 描述 属性值
    border-style 边框风格 none，solid(实线)，dashed(虚线)，dotted(点线)，double(双实线)
    border-width 边框宽度 3px
    border-color 边框颜色 black
    border 简写属性 没有数量和顺序要求

  • 盒子模型的外边距

    属性 描述 属性值
    margin-top 上外边距 3px
    margin-left 左外边距 3px
    margin-right 右外边距 3px
    margin-bottom 下外边距 3px
    margin 简写属性 3px 3px 3px 3px，3px

6.4 处理内容溢出

  • 元素内容溢出

    属性 描述 属性值
    overflow 溢出内容处理方式 visiable(显示)，hidden(隐藏)，scroll(显示滚动条)，auto(自动显示滚动条)
    overflow-x 水平方向处理方式 同上
    overflow-y 垂直方向处理方式 同上

6.5 隐藏元素

  • 隐藏元素
    • visibility属性：设置为 hidden 占有原始位置但不显示
    • display属性：设置为 none 不占用位置也不显示

第七节 浮动

7.1 浮动简介

  • 浮动：最初用来实现文字环绕图片效果的，现在浮动是主流的页面布局方式之一
  • 元素浮动的特点
    • 脱离文档流
    • 浮动后的元素默认宽和高尽可能小
    • 可以和其他元素共占一行

7.2 消除浮动的影响

  • 元素浮动的影响

    • 后面的兄弟元素会占据浮动元素之前的位置
    • 不能撑起父元素的高度，导致父元素高度塌陷

  • 消除浮动的影响(任意选择一个)

    • 给父元素指定高度

    • 给父元素也设置浮动

    • 给父元素设置 overflow:hidden

    • 在所有浮动元素的最后面，添加一个块级元素，并给该块级元素设置 clear:both

    • 给浮动元素的父元素，设置伪元素，通过伪元素清除浮动

第八节 定位

8.1 相对定位

  • 设置相对定位
    • 参考点为自己原来的位置
    • 给元素设置 position:relative 实现相对定位
    • 使用 left，right，top，bottom 四个属性调整位置
  • 相对定位的特点
    • 不会脱离文档流，不会对其他元素产生影响
    • 定位的显示元素比普通元素层次高

8.2 绝对定位

  • 设置绝对定位
    • 参考点为自己的包含块，即父元素或整个页面
    • 给元素设置 position:absolute 实现绝对定位
    • 使用 left，right，top，bottom 四个属性调整位置
  • 绝对定位的特点
    • 会脱离文档流，会对其他元素产生影响
    • 绝对定位、浮动不能同时设置，如果同时设置，浮动失效，以定位为主
    • 无论是什么元素（行内、行内块、块级）设置为绝对定位之后，都变成了定位元素

8.3 固定定位

  • 设置固定定位
    • 参考点为自己的视口
    • 给元素设置 position:fixed 实现固定定位
    • 使用 left，right，top，bottom 四个属性调整位置
  • 固定定位的特点
    • 会脱离文档流，会对其他元素产生影响
    • 固定定位和浮动不能同时设置，如果同时设置，浮动失效，以固定定位为主
    • 无论是什么元素（行内、行内块、块级）设置为固定定位之后，都变成了定位元素

8.4 粘性定位

  • 设置粘性定位
    • 参考点为最近的一个拥有滚动机制的祖先元素
    • 给元素设置 position:sticky 实现固定定位
    • 使用 left，right，top，bottom 四个属性调整位置
  • 粘性定位的特点
    • 不会脱离文档流，使用一种专门用于窗口流动的定位方式
    • 最常用的值是 top 值

8.5 定位的特殊应用

  • 定位层次
    • 定位元素的显示层级比普通元素高
    • 如果位置发生重叠，默认情况是：后面的元素，会显示在前面元素之上
    • 可以通过 z-index 属性调整元素的显示层级，值越大显示层级越高
  • 让定位元素充满包含块
    • 宽度与包含块保持一致，给定位元素同时设置 left 和 right 设置为0
    • 高度与包含块保持一致，给定位元素同时设置 top 和 bottom 设置为0
  • 让定位元素在包含块中居中
    • 四个定位属性都设置为0
    • margin 属性设置为 auto

第九节 布局

9.1 常用布局名词

  • 版心

    • 表示一个固定宽度且水平居中的盒子，来显示网页的主要内容

    • 宽度一般是 960 ~ 1200 像素之间

    • 可以是一个，也可以是多个

  • 常用布局名词

    位置 描述
    顶部导航条 topbar
    页头 header，page-header
    导航 nav，navigator，navbar
    搜索框 search，search-box
    横幅，广告，宣传图 banner
    主要内容 content，main
    侧边栏 aside，sidebar
    页脚 footer，page-footer

9.2 重置默认样式

  • 重置默认样式

    • 使用全局选择器

    • 使用 reset.css 清除默认样式

    • 使用 Normalize.css 在清除了一部分默认样式时，同时保护了有价值的默认样式
