第一节 Gradle概述

  • Gradle 简介：一款 Google 推出的基于 JVM、通用灵活的项目构建工具，支持 Maven，JCenter 多种第三方仓库，支持传递性依赖管理、废弃了繁杂的 xml 文件，转而使用简洁的、支持多种语言的 build 脚本文件

  • Gradle 项目目录结构

    • build：封装打包后的字节码，打成的包，测试报告等信息
    • gradle：封装包装器文件夹
    • src
      • main
        • java
        • resource
        • webapp
      • test
        • java
        • resource
    • gradlew
    • gradlew.bat：包装器启动脚本
    • build.gradle：构建脚本
    • settings.gradle：设置文件，定义项目及子项目名称信息

  • 常用指令

    • 清空 build 目录

    • 编译业务代码和配置文件

    • 编译测试代码，生成测试报告

    • 构建项目

    • 跳过测试构建项目

  • Wrapper 包装器
