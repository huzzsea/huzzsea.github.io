第一节 HTML概述

  • HTML：全称 Hyper Text Markup Language，即超文本标记语言，是用来描述网页的一种标记语言，它使用标记标签来描述网页，文档中包含了标签及其文本内容
  • HTML 标签：是由尖括号包围的关键词，通常是成对出现的，标签对中的第一个标签是开始标签，第二个标签是结束标签

  • HTML 元素：HTML 标签和 HTML 元素通常都是描述同样的意思，但是严格来讲，一个 HTML 元素包含了开始标签与结束标签

  • HTML 注释

  • HTML 文档基本结构

  • 版本声明：表示 HTML5 文档

  • 中文编码：目前在大部分浏览器中，直接输出中文会出现中文乱码的情况，这时候我们就需要在头部将字符声明为 UTF-8 或 GBK

第二节 基本标签

  • html 标签：整个网页是从 <html> 这里开始的，然后到 </html> 结束

  • head 标签：head 标签代表页面的头，定义一些特殊内容，这些内容往往都是不可见内容（在浏览器不可见）

    标签 描述
    <title> 定义网页的标题
    <meta> 定义网页的基本信息（供搜索引擎）
    <style> 定义 CSS 样式
    <link> 链接外部 CSS 文件或脚本文件
    <script> 定义脚本语言
    <base> 定义页面所有链接的基础定位

  • body 标签：body 标签代表页面的身，定义网页展示内容，这些内容往往都是可见内容（在浏览器可见）

第四节 段落与文字

  • 段落与文字标签

    标签 描述
    <h1>~<h6> 标题
    <p> 段落
    <br> 换行
    <hr> 水平线
    <div> 分割
    <span> 区域

  • 文本格式化标签

    标签 描述
    <b> 粗体
    <em> 着重
    <i> 斜体字
    <small> 小号字
    <strong> 加重语气
    <sub> 下标字
    <sup> 上标字
    <ins> 插入字
    <del> 删除字

  • 网页特殊符号

    标签 描述
    &nbsp 空格
    &lt 小于
    &gt 大于
    &le 小于等于
    &ge 大于等于

  • 自闭合标签：只有开始符号而没有结束符号，例如 <br/> 和 <hr/>

  • 块元素和行内元素

    • 块元素特点：独占一行，排斥其他元素跟其位于同一行，块元素内部可以容纳其他块元素或行元素，例如<h1~h6>、<p>、<hr>、<div>
    • 行内元素特点：可以与其他行内元素位于同一行，行内内部可以容纳其他行内元素，但不可以容纳块元素，例如<strong>、<em>、<span>

第七节 图像

  • 图像标签：使用标签 <img> 来设置图像

    属性 描述
    src 图像地址
    alt 无法显示图像时的提示文字
    title 鼠标移到图像上时的提示文字

  • 图像格式

    • JPG：可以很好处理大面积色调的图像，如相片、网页一般的图片
    • PNG：图片体积小，而且无损压缩，能保证网页的打开速度，支持透明信息
    • GIF：图像效果很差，但是可以制作动画

第八节 链接

  • 链接标签：使用标签 <a> 来设置超文本链接

  • href 属性：超链接所指向的 URL

  • target 属性：定义被链接的文档在何处显示

    属性值 描述
    _self 默认方式，在当前窗口打开链接
    _blank 在一个全新的窗口打开链接
    _top 在顶层框架中打开链接
    _parent 在当前框架的上一层里打开链接

      • id属性

第九节 区块

  • div标签：可用于组合其他 HTML 元素的容器，会自动换行

  • span标签：可用作文本的容器，不会自动换行

第十节 表单

  • 表单标签：使用标签 <form> 来设置表单

  • action 属性：处理表单提交的 URL

  • 输入控件标签：使用标签 **<input>**来设置输入控件

    • 文本输入框text：输入文本

      • name 属性：数据的名称

      • value 属性：输入框的默认输入值

      • maxlength 属性：输入框最大可输入长度

    • 密码输入框password：输入密码

      • name 属性：数据的名称

      • value 属性：输入框的默认输入值

      • maxlength 属性：输入框最大可输入长度

    • 单选框radio：选择一个属性

      • name 属性：数据的名称，多个 radio 的 name 属性值要保持一致

      • value 属性：提交的数据值

      • checked 属性：让该单选按钮默认选中

    • 复选框checkbox：选择多个属性

      • name 属性：数据的名称

      • value 属性：提交的数据值

      • checked 属性：让该复选框默认选中

    • 隐藏域hidden：用户不可见的输入区域，用于携带固定数据

      • name 属性：指定数据的名称

      • value 属性：指定的是真正提交的数据

    • 提交按钮submit：提交表单

      • value 属性：指定按钮文字

    • 重置按钮reset：重置表单数据，使用 button 标签生成按钮

      • value 属性：指定按钮文字

    • 普通按钮button：绑定特定事件

      • value 属性：指定按钮文字

    • 文本域textarea：输入文本框，使用 textarea 标签生成文本域

      • rows 属性：指定默认显示的行数，会影响文本域的高度

      • cols 属性：指定默认显示的列数，会影响文本域的宽度

    • 下拉框option：下拉选择属性，使用 select 和 option 嵌套标签

      • name 属性：指定数据的名称

      • value 属性：提交数据的值

      • selected 属性，表示默认选中

  • 禁用表单控件：给表单控件设置禁用属性

  • label标签：与表单控件相关联，点击后可以获取焦点

    • 设置for属性

    • 嵌套label标签

第十一节 框架

  • iframe标签：在网页中嵌入其他文件

    • src属性：框架引用的网址

    • name属性：框架名称

    • width属性：框架的宽度

    • height属性：框架的高度

    • frameborder属性：是否显示边框，1表示是，0表示否

第十二节 多媒体

  • video标签：定义视频
    • src 属性：视频的地址
    • width 属性：播放器的宽度
    • height 属性：播放器的高度
    • preload 属性：是否预加载视频
  • audio标签：定义音频
    • src 属性：音频的地址
    • preload 属性：是否预加载音频
