第一节 JSON简介

  • JSON：全称是 JavaScript Object Notation，即 JavaScript对象标记法，是一种轻量级的数据交换格式，它既易于人阅读和编写，也易于机器解析和生成，而且相比 XML 文件更小，因此迅速成为网络上十分流行的交换格式

第二节 JSON对象定义和基本使用

  • 语法规则

    • 数组 Array 用方括号 [] 表示
    • 对象 0bject 用大括号 {} 表示
    • 名称/值对 name/value 组合成数组和对象
    • 名称 name 置于双引号中，值 value 有字符串、数值、布尔值、null、对象和数组
    • 并列的数据之间用逗号 , 分隔

  • JSON的定义

第三节 JSON的解析和生成

  • **JSON.parse()**：将 JSON 字符串转换为 JS 对象

  • **JSON.stringify()**：将 JS 对象转换为 JSON 字符串
