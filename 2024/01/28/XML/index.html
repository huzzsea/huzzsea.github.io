第一节 XML简介

  • XML简介：全称为 Extensible Markup Language，即可扩展标记语言，是万维网协会推出的一套数据交换标准，被设计用来传输和存储数据，可用于定义 Web 网页上的文档元素以及复杂数据的表述和传输
  • XML的特点
    • XML 被设计用来传输和存储数据，其焦点在于数据的内容
    • XML 标签没有被预定义，需要自行定义标签
    • XML 不会替代 HTML，在 Web 应用程序中，XML 用于传输数据，而 HTML 用于格式化并显示数据

第二节 XML语法规则

  • XML文档声明

    • version：版本规范声明
    • encoding：文档字符编码
    • standalone：文档定义是否独立使用

  • XML元素

    • 每个 XML 文档必须有且只有一个根元素

    • XML 元素指的是 XML 文件中出现的标签，一个标签分为开始标签和结束标签

    • 一个标签中也可以嵌套若干子标签

    • 对于 XML 标签中出现的所有空格和换行，XML 解析程序都会当做标签内容进行处理，下列内容有不同的意义

    • 命名规范：一个 XML 元素可以包含字母、数字以及其它一些可见字符，但必须遵守以下规范

      • 区分大小写
      • 不能以数字或下划线 _ 开头
      • 元素内不能包含空格
      • 名称中间不能包含冒号 :
      • 可以使用中文，但一般不这么用

  • XML属性

    • 属性值用双引号或单引号分隔

    • 一个元素可以有多个属性

    • 预定义实体字符

      实体字符 含义
      &lt <
      &gt >
      &amp &
      &apos ‘
      &quot “

  • XML注释

  • XML CDATA节：避免让解析引擎解析执行，而是将数据当做原始内容处理，不会因为非法字符而报错

  • XML处理指令：用来指示解析引擎如何解析 XML 文件
